# AgentFlow - Agentic Workflow Platform

An AI-driven workflow automation platform that helps manage tasks, content, and agents.

## Features

- **Workflow Management**: Create and manage automated workflows with triggers, actions, and conditions
- **AI Agents**: Deploy specialized AI agents for content creation, task management, and data analysis
- **Real-time Dashboard**: Monitor workflow status and agent performance
- **Intuitive Interface**: Clean, modern UI inspired by leading workflow platforms

## Quick Start

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Architecture

- **Frontend**: Next.js 14 with TypeScript and Tailwind CSS
- **Components**: Modular React components for workflows and agents
- **State Management**: React hooks for local state management
- **Styling**: Tailwind CSS with responsive design

## Core Components

- `WorkflowCard`: Displays individual workflow information and controls
- `AgentCard`: Shows AI agent status and capabilities
- Dashboard with real-time statistics and management interface
